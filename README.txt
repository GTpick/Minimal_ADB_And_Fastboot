This repository will hold the source code for the **Minimal ADB and Fastboot** (MAF) tool.
More information about Minimal ADB and Fastboot can be found at the link below
on xda-developers:

http://forum.xda-developers.com/showthread.php?t=2317790

In this respository you the source code for older versions of MAF as well as the source code
for the automated build script designed to generate nightly builds.
